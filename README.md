その後、画面の指示に従うと、設定が完了します。
# private repositoryへの接続
private repositoryへの接続はsshを使用するため、git@github.comから始まるアドレスを使用します。<br>

git clone git@github.com:~<br>
を実行することによってカレントディレクトリにレポジトリをクローン出来ます。<br>
